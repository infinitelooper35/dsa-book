# Page 59 — Continuous Subarray Sum: Divisible by K with Length Constraint

Yesterday’s page covered the **longest** subarray whose sum is divisible by `k`. A very common interview follow-up is this boolean variant:

Given `nums` and `k`, return `True` if there exists a subarray of length **at least 2** whose sum is a multiple of `k`.

At first glance it looks similar—but there’s a crucial twist: you only need existence, and the minimum length rule changes the hash-map logic.

## Core idea

Use prefix sums modulo `k` again.

If two prefix sums have the same remainder, their difference is divisible by `k`. So if remainder `r` appears at indices `j` and `i`, then subarray `(j+1..i)` has sum divisible by `k`.

For this problem, that subarray must have length at least 2:

`i - j >= 2`

So store the **first index** where each remainder was seen. Then when the same remainder appears again, check the index gap.

## Interview-grade Python

```python
def check_subarray_sum(nums, k):
    if k == 0:
        # Need length >= 2 and sum == 0
        # With integer inputs, this means two consecutive zeros
        for i in range(1, len(nums)):
            if nums[i] == 0 and nums[i - 1] == 0:
                return True
        return False

    first = {0: -1}  # remainder 0 before array starts
    pref = 0

    for i, x in enumerate(nums):
        pref += x
        r = ((pref % k) + k) % k

        if r in first:
            if i - first[r] >= 2:
                return True
        else:
            # keep earliest index only
            first[r] = i

    return False
```

Complexity:
- Time: `O(n)`
- Space: `O(min(n, |k|))`

## Why “first index only” matters

If you overwrite a remainder with a newer index, you might lose a valid longer gap that satisfies `>= 2`.

Example:
- remainders at indices: `r` seen at `1`, then at `3`
- gap is `2` (valid)
- if you overwrote `1` with `2`, then at `3` gap becomes `1` (invalid), and you miss the answer.

## What to say out loud in interviews

“I’ll use prefix remainder matching. Equal remainders imply divisible subarray sums. I store each remainder’s first index, and when it repeats I check whether the distance is at least 2. That gives linear time.”

Also call out edge cases proactively:
1. `k = 0` needs special handling.
2. Negative numbers are safe with normalized modulo.
3. Seed `remainder 0 -> index -1` to allow subarrays starting at index 0.

Pattern memory: when you hear **‘multiple of k’ + ‘exists’ + ‘min length’**, think **remainder first-index map + distance check**.
