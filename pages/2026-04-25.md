# Page 60 — Count Subarrays Divisible by K: Frequency Map Pattern

Yesterday you solved the boolean version (“does any valid subarray exist?”). Today’s upgrade is the counting version:

Given `nums` and `k`, return how many subarrays have sum divisible by `k`.

This is a favorite because it tests whether you can go beyond detection and reason about combinations.

## Core shift: from first index to frequency

For existence, you stored the first index of each remainder.
For counting, you store how many times each remainder has appeared.

Why? If current prefix remainder is `r`, then every previous prefix with remainder `r` forms one valid subarray ending here.

So when you see `r` again:
- add `freq[r]` to answer
- then increment `freq[r]`

That’s the whole pattern.

## Interview-grade Python

```python
from collections import defaultdict


def subarrays_div_by_k(nums, k):
    freq = defaultdict(int)
    freq[0] = 1  # empty prefix

    pref = 0
    ans = 0

    for x in nums:
        pref += x
        r = ((pref % k) + k) % k  # normalize for negatives

        ans += freq[r]   # all previous same-remainder prefixes
        freq[r] += 1

    return ans
```

Complexity:
- Time: `O(n)`
- Space: `O(min(n, |k|))`

## Why this works (quick interview proof)

Let `P[i]` be prefix sum up to index `i`.
Subarray sum `(j+1..i) = P[i] - P[j]`.
This is divisible by `k` iff:

`P[i] % k == P[j] % k`

So for each `i`, number of valid `j` values is exactly the count of prior prefixes with same remainder. Frequency map gives that in O(1) average time per element.

## Common mistakes to avoid

1. **Forgetting `freq[0] = 1`**
   Then you miss subarrays starting at index 0.

2. **Using first index map here**
   That solves longest/existence flavors, not count.

3. **Not normalizing modulo with negatives**
   In Python `%` is non-negative for positive `k`, but normalized form is portable and interview-safe.

4. **Updating frequency before using it**
   Do `ans += freq[r]` first, then `freq[r] += 1`.

## What to say out loud

“I’ll use prefix remainders with a frequency map. Each time a remainder repeats, it pairs with all previous occurrences to form divisible subarrays. So I accumulate `freq[r]` into the answer in one pass.”

Pattern memory:
- **Exists?** first index + distance check
- **Longest?** first index + max span
- **Count?** frequency map + combinational accumulation

Same mathematical backbone, three interview variants.
