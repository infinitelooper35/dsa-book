# Page 57 — Subarray Sum Divisible by K (Normalize Modulo, Then Count Pairs)

Let’s stay on the prefix-sum track, but move to a classic trap-heavy interview variant:

Given an integer array `nums` and integer `k`, count subarrays whose sum is divisible by `k`.

Most misses happen for one reason: candidates remember prefix sums, but forget modulo behavior with negatives.

## Core idea

If two prefix sums leave the same remainder when divided by `k`, their difference is divisible by `k`.

Let running prefix be `pref`. For each index, compute:

`r = pref % k`

Any prior prefix with the same remainder `r` forms a valid subarray ending here.

So we track frequencies of remainders:
- `freq[r]` = how many times remainder `r` has appeared so far
- answer increment = `freq[r]` before adding current one

Seed with `freq[0] = 1` so subarrays starting at index 0 are counted.

## Interview-grade Python

```python
from collections import defaultdict

def subarrays_div_by_k(nums, k):
    freq = defaultdict(int)
    freq[0] = 1

    pref = 0
    ans = 0

    for x in nums:
        pref += x
        r = pref % k
        # Language-safe normalization pattern:
        # r = ((pref % k) + k) % k

        ans += freq[r]
        freq[r] += 1

    return ans
```

## Why modulo normalization matters

In Python, `%` is already non-negative for positive `k`, so `pref % k` is fine.
In C++/Java, negative prefixes can produce negative remainders. Use:

`r = ((pref % k) + k) % k`

Say this out loud in interviews if language is C++/Java — it’s a strong “I know edge-case arithmetic” signal.

## Quick walkthrough

`nums = [4, 5, 0, -2, -3, 1], k = 5`

Remainders while scanning: `4, 4, 4, 2, 4, 0`

Every time remainder repeats, we add existing count:
- second `4` adds 1
- third `4` adds 2
- next `4` adds 3
- final `0` adds 1 (from seed)

Total = 7.

## What to say in the interview

“I’ll use prefix sums modulo `k`. If two prefixes have equal remainder, their difference is divisible by `k`, so that subarray qualifies. I’ll count prior equal remainders with a hashmap. Time `O(n)`, space `O(k)` in bounded remainder space, otherwise `O(n)`.”

Pattern memory: **divisible subarray count** usually means **prefix sum + remainder frequency map**.
